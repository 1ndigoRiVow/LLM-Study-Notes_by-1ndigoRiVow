# 矩阵分解：从特征分解到 SVD

矩阵分解是线性代数在机器学习（尤其是 LLM 降维与压缩）中最重要的应用之一。本章我们将理清特征分解与奇异值分解（SVD）的核心逻辑。

---

## 1. 特征分解 (Eigendecomposition)

特征分解是将一个**方阵**分解为特征向量和特征值的过程。

对于 $n \times n$ 的方阵 $A$，如果它有 $n$ 个线性无关的特征向量，则可以分解为：

$$A = Q \Lambda Q^{-1}$$

- **$Q$**：特征向量矩阵，每一列是一个特征向量。
- **$\Lambda$**：对角矩阵，对角线上的元素是对应的特征值 $\lambda_i$。

::: warning 局限性
特征分解要求矩阵必须是**方阵**。但在处理 LLM 的权重矩阵或用户-行为矩阵时，矩阵往往是长方形的（$m \times n$），此时我们需要更通用的 SVD。
:::

## 2. 奇异值分解 (SVD)

SVD 可以对任意形状的矩阵进行分解，其数学表达式为：

$$A = U \Sigma V^T$$

- **$U$** ($m \times m$)：左奇异向量，对应 $AA^T$ 的特征向量。
- **$\Sigma$** ($m \times n$)：奇异值矩阵，对角线上的值 $\sigma_i$ 是奇异值（按降序排列）。
- **$V^T$** ($n \times n$)：右奇异向量的转置，对应 $A^TA$ 的特征向量。

### 核心直觉
- 奇异值 $\sigma$ 衡量了该维度信息的“重要程度”。
- 在模型压缩中，我们只保留前 $k$ 个最大的奇异值，从而实现**低秩近似**（Low-Rank Approximation）。

## 3. Python 实践

利用 `NumPy` 我们可以非常轻松地完成这个过程：

```python
import numpy as np

# 定义一个 3x2 矩阵
A = np.array([[1, 2], 
              [3, 4], 
              [5, 6]])

# 执行 SVD 分解
U, S, Vt = np.linalg.svd(A)

print(f"左奇异矩阵 U 形态: {U.shape}")
print(f"奇异值 S: {S}")
print(f"右奇异矩阵 Vt 形态: {Vt.shape}")
````
