# 矩阵分解：特征分解，奇异值分解SVD

矩阵分解是线性代数在机器学习（尤其是 LLM 降维与压缩）中最重要的应用之一。本章理清特征分解与奇异值分解（SVD）的核心逻辑。

---

## 0. 特征值和特征向量

**定义**：若对于 $N$ 阶矩阵 $A$，存在实数 $\lambda$ 及非零向量 $x$，满足 $Ax = \lambda x$，则称 $\lambda$ 是 $A$ 的特征值，非零向量 $x$ 是 $A$ 的特征向量。

**求解**：求解特征值和特征向量的过程就是**齐次线性方程组求非零解**的过程：

$(A - \lambda E)x = 0$。

齐次线性方程组若要存在非零解，那么需要系数行列式 $|A - \lambda E| = 0$，
即系数矩阵 $r(A - \lambda E) < n$。

::: 求矩阵的秩
初等行变换法化为行阶梯形矩阵，非零行个数即为矩阵的秩。
:::

::: 求矩阵的逆
1. 伴随矩阵法
若 $|A| \neq 0$，则：$$A^{-1} = \frac{A^*}{|A|}$$

2. 初等变换法设 $A$ 为可逆阵，通过初等行变换实现：
$$(A \mid E) \xrightarrow{\text{初等行变换}} (E \mid A^{-1})$$
:::

## 1. 特征分解 (ED)

特征分解是将一个**方阵**分解为特征向量和特征值的过程。

对于 $n$ 阶方阵 $A$，如果它有 $n$ 个线性无关的特征向量，则可以分解为：

$$A = Q \Lambda Q^{-1}$$

- **$Q$**：特征向量矩阵，每一列是一个特征向量。
- **$\Lambda$**：对角矩阵，对角线上的元素是对应的特征值 $\lambda_i$。

::: warning 局限性
特征分解要求矩阵必须是**方阵**。但在处理 LLM 的权重矩阵或用户-行为矩阵时，矩阵往往是长方形的（$m \times n$），此时需要更通用的 SVD。
:::

## 2. 奇异值分解 (SVD)

SVD 可以对任意形状的矩阵进行分解，其数学表达式为：

$$A = U \Sigma V^T$$

- **$U$** ($m \times m$)：左奇异向量，对应 $AA^T$ 的特征向量。
包含了有关行的所有信息（代表自己的观点）。

- **$\Sigma$** ($m \times n$)：奇异值矩阵，对角线上的值 $\sigma_i$ 是奇异值（按降序排列）。
记录SVD过程（是一种日志）。

- **$V^T$** ($n \times n$)：右奇异向量的转置，对应 $A^TA$ 的特征向量。
包含了有关列的所有信息（代表自己的特征）。

### 核心直觉
- 矩阵的物理意义就是空间变换，SVD的物理意义就是分解 —— 将一个复杂的转换过程分解为三个简单的基本变换（旋转、伸缩、旋转）。奇异值 $\sigma$ 的大小表征了相应奇异向量的伸缩程度（可以理解为权重），奇异值越大，对应的奇异向量对最终结果影响越大。奇异值矩阵有几个非零奇异值，则矩阵A就是秩几矩阵。（可用于图像压缩、去噪）。
- 在模型压缩中，我们只保留前 $k$ 个最大的奇异值，从而实现**低秩近似**（Low-Rank Approximation）。

### 应用
- 降维：（左奇异矩阵用于行数压缩；右奇异矩阵用于列数/维数压缩）
- 图像压缩
- 图像去噪

## 3. Python 实践

利用 `NumPy` 我们可以非常轻松地完成这个过程：

```python
import numpy as np

# 定义一个 3x2 矩阵
A = np.array([[1, 2], 
              [3, 4], 
              [5, 6]])

# 执行 SVD 分解
U, S, Vt = np.linalg.svd(A)

print(f"左奇异矩阵 U 形态: {U.shape}")
print(f"奇异值 S: {S}")
print(f"右奇异矩阵 Vt 形态: {Vt.shape}")
```
